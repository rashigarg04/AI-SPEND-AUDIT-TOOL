# Pricing Dataset

## Included AI Platforms

- ChatGPT
- Claude
- Cursor
- GitHub Copilot
- Gemini
- OpenAI API
- Anthropic API
- Windsurf

---

## Data Structure

Each tool contains:
- Plan name
- Monthly cost
- Feature category
- Seat support

---

```js
ChatGPT: {
