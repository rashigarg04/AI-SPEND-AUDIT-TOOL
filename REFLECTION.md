# Reflection

This project helped me understand the development lifecycle of a SaaS product from idea to deployment.

I learned:
- Full-stack application development
- Firebase integration
- Authentication systems
- State management
- Dashboard design
- API route handling
- Cloud deployment

Challenges faced:
- Dependency conflicts
- Next.js version issues
- Firebase SSR problems
- Dynamic chart rendering

Solutions:
- Reconfigured dependencies
- Used client-side rendering
- Modularized components
- Improved state handling

This project improved both my frontend and backend development skills.
