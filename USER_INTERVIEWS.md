# User Interviews

## Interview 1

User Type: Startup Founder

Pain Point:
Managing multiple AI subscriptions across teams.

Desired Feature:
Centralized AI cost visibility.

---

## Interview 2

User Type: Developer

Pain Point:
Using multiple overlapping coding assistants.

Desired Feature:
Plan recommendations.

---

## Interview 3

User Type: Agency Team

Pain Point:
Tracking team AI spending.

Desired Feature:
Dashboard analytics and exports.

## Key Insights

- Users want cost transparency
- Teams use overlapping AI tools
- Savings recommendations are valuable
